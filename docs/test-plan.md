# 功能测试计划

> VisuTask 各模块功能测试规划

---

## 测试策略

| 层级 | 范围 | 工具 | 说明 |
|------|------|------|------|
| 单元测试 | Go 后端各模块函数 | `testing` + `testify` | 每个模块独立测试 |
| 集成测试 | 模块间协作、Wails IPC | `testing` + mock | 模拟 Wails 环境 |
| E2E 测试 | 完整任务流程 | 手动验证 | 端到端验证 |

---

## 1. Agent 模块测试

### 1.1 Parser（任务解析器）

| 测试项 | 输入 | 预期输出 | 优先级 |
|--------|------|----------|--------|
| 简单任务解析 | "打开浏览器访问 GitHub" | Intent{Action: "操作", Target: "浏览器"} | P0 |
| 多步骤任务 | "打开 Excel 复制数据到 CRM" | 包含多步操作的 Intent | P0 |
| 带参数任务 | "录入 100 条数据" | Intent{LoopCount: 100} | P0 |
| 模糊描述处理 | "帮我弄一下那个东西" | 返回需追问的提示 | P1 |
| 超长描述 | 1000 字以上的详细描述 | 正确提取关键信息 | P2 |
| 空输入 | "" | 返回错误 | P0 |

### 1.2 Planner（计划生成器）

| 测试项 | 输入 | 预期输出 | 优先级 |
|--------|------|----------|--------|
| 基础计划生成 | Intent + 截图 + OCR 结果 | 合理的步骤序列 | P0 |
| 步骤顺序正确性 | 多步骤任务 | 步骤依赖关系正确 | P0 |
| 目标定位准确性 | "点击保存按钮" + 含"保存"的 OCR | Step.TargetOCR 匹配 | P0 |
| 置信度评估 | 明确目标 vs 模糊目标 | 明确目标置信度 > 模糊目标 | P1 |
| 无匹配目标 | 目标文字不在 OCR 结果中 | Confidence < 0.5 或标记失败 | P0 |

### 1.3 Simulator（模拟演示器）

| 测试项 | 预期行为 | 优先级 |
|--------|----------|--------|
| 截图返回 | 返回有效的 base64 PNG | P0 |
| 标注覆盖图 | 目标区域有高亮框 | P0 |
| 不执行实际操作 | 模拟后屏幕状态不变 | P0 |
| 置信度返回 | 返回 0-1 的 float64 | P0 |
| 坐标准确性 | TargetRect 与实际目标位置匹配 | P1 |

### 1.4 Executor（步骤执行器）

| 测试项 | 预期行为 | 优先级 |
|--------|----------|--------|
| click 执行 | 鼠标移动到目标位置并点击 | P0 |
| input 执行 | 定位输入框并输入文字 | P0 |
| verify 执行 | 检查目标文字是否出现 | P0 |
| 超时处理 | 超过 timeout 后标记失败 | P0 |
| 进度事件推送 | 每步执行时 EmitEvent | P0 |
| 错误事件推送 | 执行失败时 EmitEvent("step:error") | P0 |
| 中止执行 | StopTask 后立即停止 | P0 |

### 1.5 Recovery（异常恢复）

| 测试项 | 场景 | 预期行为 | 优先级 |
|--------|------|----------|--------|
| 自动重试 | 单次执行失败 | 重试 N 次（可配置） | P0 |
| 回退操作 | 连续失败 | 回退到上一步重新执行 | P1 |
| 人工介入 | 重试耗尽 | 弹出提示等待用户操作 | P0 |
| 界面变化 | 执行中目标位置变化 | 重新 OCR 定位 | P1 |

## 7. 端到端流程测试

### 7.1 任务创建流程

| 步骤 | 操作 | 预期结果 | 优先级 |
|------|------|----------|--------|
| 1 | 输入任务描述 | 文本框正常显示 | P0 |
| 2 | 点击"生成计划" | 返回步骤列表 | P0 |
| 3 | 点击"模拟演示" | 截图 + 标注目标显示 | P0 |
| 4 | 点击"确认此步" | 进入下一步 | P0 |
| 5 | 确认所有步骤 | 显示步骤总览 | P0 |
| 6 | 点击"保存任务" | 任务出现在任务列表中 | P0 |

### 7.2 任务执行流程

| 步骤 | 操作 | 预期结果 | 优先级 |
|------|------|----------|--------|
| 1 | 在任务列表选择任务 | 任务详情加载 | P0 |
| 2 | 点击"执行" | 开始逐步执行 | P0 |
| 3 | 观察进度 | 实时显示步骤进度 | P0 |
| 4 | 执行完成 | 显示结果报告 | P0 |
| 5 | 查看日志 | 执行记录出现在日志中 | P0 |

### 7.3 异常流程

| 场景 | 预期行为 | 优先级 |
|------|----------|--------|
| 执行中目标消失 | 重试 → 失败 → 提示用户 | P0 |
| 执行中用户中止 | 立即停止，已执行步骤记录保留 | P0 |
| LLM 服务不可用 | 提示用户检查配置 | P0 |
| OCR 引擎未安装 | 提示安装引导 | P0 |
